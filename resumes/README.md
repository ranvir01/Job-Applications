## Usage

1. **Master Resume**: Keep this fully updated with ALL experiences
2. **Role-Specific Versions**: Use these as starting points when applying
3. **Application-Specific**: Create further tailored versions in each application folder

## Tips

- Always update master resume first, then update role versions
- Include keywords from job descriptions naturally
- Quantify achievements with numbers where possible
- Use strong action verbs
- Keep to 1 page for early career, 2 pages max for experienced
